# CLSVR Homepage

CLSVRの静的ホームページです。`index.html` をトップページとして、Vercelでそのまま公開できます。

## 構成

- `index.html`: トップページ
- `styles.css`: 画面デザインとレスポンシブ対応
- `script.js`: LINE予約文面生成、スクロール演出
- `assets/`: サイト内で使うトリミング済み画像
- `vercel.json`: Vercel静的配信用の設定

## Vercel Deploy

VercelのGitHub連携を使う場合は、このフォルダをGitHubリポジトリにアップし、VercelでImportしてください。

Framework Presetは `Other`、Build Commandは空、Output Directoryも空のままで公開できます。
