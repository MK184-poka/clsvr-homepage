# CLSVR image rules

Use lowercase English file and folder names only.

- `entrance/`: storefront and door images. `entrance-main.png` is the closed first-door image and `entrance-open.png` is the opened-door transition image.
- `inside/`: shop interior and about images. `shop-inside.png` is the current web shop interior image.
- `car/`: service car and travel images. `clsvr-car.png` is the current service car image.
- `services/`: service card images for massage, lifestyle support, wasp removal, and web support.
- `ui/`: signs, paper, and decorative UI images

Recommended production format is WebP. Keep the same role-based names when replacing images, then update the file extension in `index.html`, `styles.css`, and `script.js`.
